Check Your Font

Fonts are ornery beasts, different in different browsers, on different platforms and at different sizes. It's a good idea to check that the family and size you want is legible on all browser-platform combinations you care about; there can be all sorts of problems hiding in the details.

Show me

To get started, check out some of these font checks:

  • Helvetica at default font sizes
  • Courier at 14, 16, 18 and 20 pixels
  • Verdana and 20 and 30 pixels, highlighting the 20 pixel version
  • Georgia showing lorem ipsum instead of an alphabetic display
  • Arvo from Google Fonts

How do I use it?

Check Your Font allows you to construct a URL to control a number of aspects of font selection. You start with the font family, so if you want to use MyFont use {{ baseurl }}MyFont. If you need to use spaces in a font name, just replace them with a plus sign (+), such as {{ baseurl }}Courier+New.

Everything else you tack on after that, separating different controls using a semicolon (;). Try one or more of the following:

;sizes=
allows you to give a comma-separated list of pixel font sizes, such as {{ baseurl }}Courier;sizes=14,16,18,20 (the default is every size from 10 pixels to 24 pixels)
;sizes.key=
allows you to set the 'key' font sizes in the same way, such as {{ baseurl }}Verdana;sizes=20,30;sizes.key=20—you can use sizes.key without setting sizes too (the default is 12, 14, 18 and 24 pixels)
;sizes.max=
;sizes.min=
is a quick way of setting the sizes to use, such as {{ baseurl }}Verdana;sizes.min=20;sizes.max=30
;text
switches to display lorem ipsum instead of the alphabet, numbers and common punctuation, such as {{ baseurl }}Georgia;text
{% comment %}
;font.font-name=url
allows you to load in a webfont from a given URL (currently we only support WOFF)
{% endcomment %}
;font.font-name=google/font-weights
allows you to load in a font from Google Fonts, such as {{ baseurl }}Arvo;font.Arvo=google/700 (font-name is the font-family name you passed in the font stack; font-weights is the list of weights you want to pull from Google)
